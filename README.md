# 👤 User Finder App

A simple and responsive **User Finder Web Application** built using **HTML, CSS, and JavaScript**.
This project fetches random user data from an external API and displays it in a clean UI.

---

## 🚀 Features

* 🔍 Fetch random users using API
* 👤 Display user name, image, and basic details
* ⚡ Fast and lightweight
* 📱 Responsive design (mobile + desktop)
* 🎯 Clean and simple UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* API: https://randomuser.me/api/

---

## 📦 API Used

We are using the following API to fetch user data:

https://randomuser.me/api/?results=7

* Returns 7 random users
* Includes profile image, name, location, etc.

---

## 📱 Responsive Design

The app is fully responsive and works on:

* Desktop screens
